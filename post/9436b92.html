🌸 语法点回顾

🙋‍♀️ 初めまして（一）

「判断文はんだんぶん」，是指以 「名词 + です」 为基本形式，用来对人、事、物进行说明、判断、归类或确认的句子。在很多教材中，也称为名词句或名词述语句。

⚠️ 注意：这里的"判断"并不是"思考、推理"的意思，而是"对人或事物的身份、类别、性质等进行说明或确认"。

🗂️ 类型 🎯 作用 📚 举例 🇨🇳 中文解释
💬 说明 告诉别人事实 私わたしは 学生がくせいです。 我是学生（说明身份）
✅ 断定 明确肯定事实 あの人ひとは 医者いしゃです。 那个人是医生（下结论）
📚 归类 表示所属类别 犬いぬは 動物どうぶつです。 狗是动物（分类说明）
📚 语法 📐 格式 💡 含义 🔑 简记 🗣️ 例句
肯定句 名词［主题］は 名词［说明］です。 用于肯定判断，说明主题是什么或属于什么类别。 ～ 是 ～ 私わたしは　留学生りゅうがくせいです。
否定判断句 名词［主题］は 名词［说明］ではありません。 用于否定判断，口语中常说成「じゃありません」。 ～ 不是 ～ 李りさんは　日本人にほんじんじゃありません。
疑问句 名词［主题］は 名词［说明］ですか。 句尾加「か」，礼貌的疑问句式，句末读升调。（⤴）。 ～ 是 ～ 吗？ 王おうさんは　中国人ちゅうごくじんですか。
所属与属性 名词［修饰语］の 名词［中心语］ 表示从属、属性等。日语中名词连接必须用「の」，不能省略。 ～ 的 ～ 私わたしの　友達ともだち

🤳 家族の写真（二）

💡 说明 💬 解释 📖 示例
👥 同一位置，面向同一方向 🎯 两人并肩站着，比如一起参观展览或看橱窗。 ✨ 视线方向一致，因此「これ／それ／あれ」的判断以“距离两人整体的远近”为基准。 👉 离我们都近的物品 これ 👈 离我们稍远的物品 それ 👀 离我们都远的物品 あれ
🤝 相隔一段距离，面对面 🎯 两人面对面交谈。两人之间存在空间分界。 ✨ 所以「これ／それ／あれ」的判断以“双方各自的立场”为基准。 ⚠️ 如果一方使用　あれ 则另一方也必须使用　あれ 📍 离自己最近的物品 これ 👂 离对方最近的物品 それ 🏫 离双方都远的物品 あれ
📚 语法 📐 格式 💡 含义 🔑 简记 🗣️ 例句
事物的指代 これ / それ / あれ［代词指示］は 名词［说明］です。 指示代词作主语，分别表示近称、中称、远称的事物。 这个/那个 是 ～ これは　辞書じしょです
连体指示修饰句 この・その・あの + 名词 は 名词 です 指示连体词，后面必须接名词，不能单独使用。 这个（物品）/那个（物品） 是 ～ この鞄かばんは　森さんのです
疑问代词 だれ/ 何なに・なん/ どれ 用于询问人、事物或在多个中选择。 谁 / 什么 / 哪个 この方はどなたですか。
年龄询问 名词［人］ は 何歳 / おいくつ ですか 询问年龄，「おいくつ」是更礼貌的尊称形式。 ～几岁/多大了？ お母かあさんは　おいくつですか

🏙️ ここはデパートです（三）

📚 语法 📐 格式 💡 含义 🔑 简记 🗣️ 例句
指示地点 ここ / そこ / あそこ［指示场所］ は 名词［场所/地点］ です。 空间指示代词作主语，用于指示具体的场所或地点。 "这里是～ / 那里是～" ここはデパートです。
询问位置 名词［寻找的对象］ は　ここ / そこ / あそこ / どこ［场所指示 / 疑问］ です（か）。 用于说明或询问某人、某物或某场所的位置。 "～在～ / ～在哪里？" トイレはどこですか。
选择疑问 名词［主题］ は　名词A ですか、名词B ですか。 选择疑问句，提供两个或多个选项供对方选择，回答时直接说出正确选项，不能用「はい/いいえ」。 是 A 还是 B 今日は水曜日ですか、木曜日ですか。
追加判断 名词も 名词です。 必须依赖前文的“垫底”，表示相同属性。 ～也是～ 私も学生です。
价格询问 名词は　いくら　ですか。 用于询问物品或服务的价格。 ～多少钱？ この鞄はいくらですか。

🏢 会社の場所（四）

📚 语法 📐 格式 💡 含义 🔑 简记 🗣️ 例句
说明位置 名词[物/人] は 名词[场所] に あります/います 说明已知事物/人的所在位置。无生命用「あります」，有生命用「います」。 “～（物/人）在～（地方）” 鞄は部屋にあります。
存在焦点 名词［场所］に 名词［物/人］ が あります／います 强调某个场所存在什么（新信息焦点）。新信息焦点必须用主格助词「が」。 “某处有某物／某人” 部屋に机があります。
名词并列 名词A と　名词B 只能连接名词，表示穷尽列举。 A 和 B 鞄と靴。
全面否定 疑问词 ＋ も ＋ 动词［否定形］ 表示毫无例外的全面否定。 “全都不 / 什么都不 / 谁都不 / 哪里都不” 部屋に誰もいません。

✨ 其他的一些语法点

🎯 助词

🎯　助词 🎵 读音 🛠️ 核心功能 💡 详细说明 📖 例句
は wa 提示主题 聚光灯💡。把名词拎出来作为谈论对象，表示“说到……”。 私 は 王です。
の no 连接名词 万能胶水🔗。表示所属、属性、机构归属或位置修饰，相当于“的”。 会社 の社員
机の上
に ni 存在地点 图钉📍。跟在场所名词后，明确标示出事物存在的具体物理坐标。 部屋にあります。
か ka 表示疑问 口头问号❓。放在句尾将陈述变为疑问。也可用于轻微感叹（读降调）。 中国人ですか。
が ga 提示新发现的主体 在存在句中，提示客观存在或新发现的事物/人。存在焦点绝不能用「は」。 机があります。
も mo 表示并列/追加 ①代替「は」表示“也”，必须建立在前面已有相同判断的基础上。 私も学生です。
も mo 全面否定标记 ②与疑问词及否定动词连用，表示“全都不”。且句尾接否定动词，表示“全都不”。 何もありません。
と to 并列（和） 专属胶水🤝。只能连接名词与名词，表示穷尽列举，相当于“A和B”。 時計と眼鏡
ね ne 确认 / 同意 寻求共鸣🤝。语调升调（↗）表示期待对方附和“……吧？”；降调表示柔和感叹。 地下鉄の駅はここです ね。
